# y-cli

A cli for front end project.

## Intro

Design for simple project.

## Installation

```bash
$ npm i -g y-cli@beta
# or
$ yarn global add y-cli@beta
```

## Usage

In your shell, run

```bash
$ y-cli
```

That's it
