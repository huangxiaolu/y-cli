# y-cli

A cli for front end project.

## Intro

Design for simple project.

## Features

For JavaScript, you are able to choose from

- React
- Vue
- Angular
- Preact
- Vanilla

When choosing Vanilla, following css pre-processors are supported

- PostCSS
- LESS
- Sass
- Stylus

## Installation

```bash
$ npm i -g y-cli@beta
# or
$ yarn global add y-cli@beta
```

## Usage

In your shell, run

```bash
$ y-cli
```

That's it
